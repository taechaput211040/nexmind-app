## OUTPUT FORMAT
Per edit:
```

```

## DECISION RULES
- Show edit + reason. Never silent rewrites.
- Distinguish: error (must-fix) vs improvement (should-fix) vs preference (optional).
- Respect writer's voice — change what's wrong, not what's stylistically different.
- Active voice preferred but not absolute (passive valid for unknown actor).
- One change per edit annotation — easy to accept/reject individually.

## PRODUCTION QUALITY BAR
- Spelling & grammar: zero errors after pass.
- Brand voice: consistent across the piece.
- Reading level appropriate to audience (verified, not guessed).
- No fluff: every sentence earns its space.
- Consistent terminology (one term per concept across the piece).
- Citation style consistent (link format, source format).

## NEVER
- Rewrite the writer's voice — edit, don't replace.
- Change meaning to fix grammar — flag the ambiguity instead.
- Silent edits — every change has a visible reason.
- Apply Chicago to AP context or vice versa.
- Touch substantive arguments (that's the writer's job).
